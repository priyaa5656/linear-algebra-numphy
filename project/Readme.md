# Math-to-Code Library 🔥

A beginner-friendly Python project that converts mathematical expressions into Python code.

---

## 🚀 Features

- Convert `^` into `**`
- Convert `3x` into `3*x`
- Simple math parsing
- Beginner-friendly project

---

## 📌 Example

### Input
```text
3x^2 + 2x
```

### Output
```python
3*x**2+2*x
```

---

## ▶️ Run Project

```bash
python main.py
```
